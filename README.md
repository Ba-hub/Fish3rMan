### Prerequisites

* gcc
  ```sh
  apt-get install gcc
  ```

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/Ba-hub/3xecutable.git
   ```
2. Compile the Script
   ```sh
   gcc -o 3xecuteable arch64_execute.c 
   ```
3. Output `3xecuteable`
   ```sh
   ./3xecuteable
   ```
